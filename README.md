

# sqc_club

Official club website repository for **Super Quant Coders (SQC)**, NIT Jalandhar.

## Description

This repository contains the source code for the official website of the Super Quant Coders (SQC) club at NIT Jalandhar.  
The website presents information about the club, its domains, projects, events, and core team.
